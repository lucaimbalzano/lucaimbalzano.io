## UI

`@simbashrd/ui` is a modern component library built with [shadcn/ui](https://github.com/shadcn-ui/ui). It provides a collection of reusable, accessible and customizable components to streamline your web development workflow.

### Installation

```bash
pnpm add @simbashrd/ui
```

### Usage

```css
@import 'tailwindcss';
@import '@simbashrd/ui/styles/preset.css';
@import '@simbashrd/ui/styles/shiki.css'; /* Optional */
@import '@simbashrd/ui/styles/typography.css'; /* Optional */
@source '../../node_modules/@simbashrd/ui/dist/**/*.js';
```

```tsx
import { Button } from '@simbashrd/ui'
```
